1.2 RabbitMQ的介绍

市面上比较火爆的几款MQ:

ActiveMQ, RocketMQ, Kafka, RabbitMQ.

  • 语言的支持：ActiveMQ，RocketMQ只支持Java语言，Kafka可以支持多门语言，RabbitMQ支持多种语言
  • 效率方面：ActiveMQ, RocketMQ, Kafka效率都是毫秒级别，RabbitMQ是微妙级别的（基于Erlang编写，基于并发的编程）。
  • 消息丢失，消息重复问题：RabbitMQ针对消息的持久化，和重复问题都有比较成熟的解决方案
  • 学习成本：RabbitMQ非常简单

RabbitMQ是由Rabbit公司去研发和维护的，最终是在Privotal.

RabbitMQ严格地遵循AMQP协议（高级消息队列协议，帮助我们在进程之间传递异步消息）

三、RabbitMQ架构

3.1 官方的简单架构

1、Publisher - 生产者：发布消息到RabbitMQ重的Exchange
2、Cosumer - 消费者：监听RabbitMQ中的Queue中的消息
3、Exchange - 交换机：和生产者连接并接收生产者的消息
4、Queue - 队列：Exchange会将消息发布到指定的Queue，Queue和消费者进行交互
5、Route - 路由：交换机以什么样的策略将消息发布到Queue

误删除硬盘（未格式化）后的重建分区表完成硬盘恢复

发表于 2021-04-21 更新于 2021-06-05

一、起因

需要帮同学在他的 ssd 中装 Windows10 操作系统，在装系统的过程中误将自己的数据盘当成同学待安装系统的 ssd。然后点了删除，就是如下图所示的那一步

若是只进行到了这一步，千万不要慌。只需要选择退出 Windows 安装，重新启动（原系统没问题的情况下），原来的非系统盘是没有被删除掉的。

但是如果之后想要再次安装系统的话，【你想将windows安装在哪里】的这个选项卡中，原来误删掉的驱动器依然会显示是删除掉的（未分配的空间）。

如果想要继续安装操作系统，务必不要嫌麻烦，先拔掉之前选择了删除的硬盘，之后再重新进安装程序。

二、表现：【位置不可用】

  1. 因为我的桌面设置过，是在 D 盘中的，所以一打开桌面就会显示【位置不可用的弹窗】

  2. 打开我的电脑，发现只有一个 C 盘系统盘

  3. 按下windows键，输入 disk 回车，能找到驱动器 1，但是空间全部为未分配

三、解决方案

  • 不要尝试重新分配空间（分区）。

重新分区后是看不到原来的数据的，因为会被格式化；而且重新分区后系统就能找到原来的 D 盘磁盘，若此时进行了一些磁盘的读写操作，磁盘就无法完全恢复了。

  • 不需要进行数据恢复

若没有进行格式化，到目前为止的操作只是 删除了分区表 ，需要做的仅仅是 重新创建分区表，而 不是进行整个磁盘的数据恢复！！！

步骤：

  1. 下载能够建立分区表的磁盘管理工具。我使用的是 DiskGeniues。缺点是 DiskGeniues 的免费版本只可以在软件中创建分区表，不能保存分区表到磁盘中。需要保存分区表需要购买【标准版】以上的版本或者使用其他手段。
  2. 选到丢失的磁盘，选择【搜索已丢失分区（重建分区表）】
  3. 重建后检查一下文件没有什么问题，就可以选择【保存分区表】
  4. 分区表格式，这个没有研究过，我选择的是【GUID】
  5. 点击确定后，硬盘就可以访问了。此时可能会遇到盘符和原来不一致的问题。若没有这一问题，就大功告成了。
  6. 若有盘符问题，打开磁盘管理（开始菜单搜索disk）,选择更改驱动器和路径，分配到原来的位置。
  7. 此时已经大功告成了，这样操作得到的数据是完全和之前一样的，不会有任何的损坏，也不需要花时间拷贝或是数据恢复，如果有一些应用或服务有问题，尝试重新启动一下计算机。

UnityBasics - 09 - Skybox and Materials

笔记

  1. 选择球体 Sphere, 将平滑度 Smoothness 调为 1, 可以观察到球体可以反射天空的材质。但并非实时运算，旁边的正方体 Cube 并没有被反射，如下图。
  2. 在 Lighting 中的三个选项：
    • Skybox Matetrial：天空球
    • Sun Source：Unity 预设的太阳（选择Default Skybox 预览）
    • Environment Lighting：环境光（最暗的颜色）
      • Source - Gradient：
        • Skycolor: 反射的天空颜色
        • Equator Color: 反射的赤道颜色
        • Ground Color: 反射的地面颜色
      • Source - Skybox:
        • Intensity Mutiplier：权重
        • Amibient Mode:实时渲染（Realtime）或烘焙（Baked）
    • Environment Reflections：环境反射光
      • source: Skybox （天空）；Custom （可以塞材质球）
      • Resolution：分辨率
      • Compression：选择Auto集显可能会出现花屏，此时需要选择Uncompressed
      • Intensity Mutiplier：权重
      • Bounces：模拟反射的次数

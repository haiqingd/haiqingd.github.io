1.UnityBasics - 01 - Installation

2021年3月12日

笔记

Unity 的安装，需要先安装 UnityHub, 这里 进入官网下载页面，选择下载 UnityHub。

在 UnityHub 中选择 激活新许可证-Unity个人版-我不以专业身份使用Unity。以个人身份使用可以免费使用4天。

如果要申请学生认证，需要关联 Github 的账号，并且在 Github 上进行学生认证，认证需要不超过23天的时间审核。之后便可以使用 Unity 个人版。
关联github
认证入口

等待认证，在 这里 可以看自己的认证进度。等待中的状态是 Pending, 认证成功后，状态是 Approved。

接下来，在 Unity 官网的学生认证界面，出现 Congratulations! 就代表认证成功了。

然后需要复制自己的学生认证序列号到 UnityHub中。
打开 Unity 账号下拉框 Settings - My Seats, 选到 Unity Student

然后勾选自己，点击 Assign Seat(s), 就能看到自己的序列号了

在 UnityHub 中添加新的许可证

大功告成！

2.UnityBasics - 02 - Windows

2021年3月14日

笔记

Unity editor layout

  • toolbar: 工作列
  • herarchy: 树状列表，表示父子关系（大纲视图）
  • scene view: 最主要的3d操作区
    • scene 视窗：关卡编辑器
    • game 视窗：玩家看到的画面
    • asset store视窗：unity的线上商城
  • inspector: 参数栏
  • project（和console视窗复用）: 资源管理器

    基本操作

  • 输出：File - build setting
  • hierarchy：create新增
  • Shift + Space：全屏当前窗口
  • project 视窗右下角可以拖拽显示大小，最小的时候就是列表形式

    疑问

  • 创建的材质可以在 inspector 视窗的右下角点击来预览

3.UnityBasics - 03/04 - Move Yourself/Move Other Object

2021年3月14日

笔记

1）移动自己的方法

  • 位移: MMB(Middle mouse button)
  • 旋转: RMB
  • 缩放: scroll MMB
  • Fly mode: RMB + W、A、S、D、Q、E
  • lock rotate(focus): f => Alt + LMB

* Create => 3D object => cube
* 复制物件 => Ctrl + D

2）移动别人的方法

  • 检视模式: Q
  • 位移: W
  • 旋转: E
  • 缩放: R
